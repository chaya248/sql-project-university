# University SQL Server Project

This project is a relational database system built using SQL Server.  
It simulates a university management system and demonstrates advanced SQL development using T-SQL.

---

## Features

- Student management system (create, update, delete, and query students)
- Lecturer management with specialization tracking
- Course management linked to lecturers and study groups
- Enrollment system connecting students to courses
- Stored procedures for validation and controlled data insertion
- User-defined functions (scalar and table-valued)
- Views for simplified data access
- Triggers for automatic timestamp updates
- Advanced SQL queries using JOIN, UNION, EXCEPT, INTERSECT, CROSS APPLY, OUTER APPLY

---

## Technologies Used

- Microsoft SQL Server
- T-SQL (Transact-SQL)

---

## Database Structure

The system includes the following tables:

- student
- lecturers
- courses
- enrollment
- circles

---

## Advanced SQL Concepts Implemented

- Stored Procedures
- Scalar and Table-Valued Functions
- Views
- Triggers
- Joins (INNER, LEFT, CROSS APPLY, OUTER APPLY)
- Set Operations (UNION, EXCEPT, INTERSECT)
- Error Handling (TRY...CATCH)
- WHILE loops

---

## Purpose of the Project

This project was developed as part of first-year studies to practice database design, relational modeling, and advanced SQL programming logic.

It demonstrates both theoretical understanding and practical implementation of SQL Server concepts.
